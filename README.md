# TaskBot

Проект состоит из двух основных файлов:

- `bot.go` - реализация телеграм-бота для управления задачами.
- `bot_test.go` - интеграционные тесты, проверяющие сценарии взаимодействия пользователей с ботом через имитацию Telegram API.

## bot.go

- запускает бота с использованием `github.com/skinass/telegram-bot-api/v5` и настраивает webhook
- обрабатывает команды `/tasks`, `/new`, `/assign`, `/unassign`, `/resolve`, `/my`, `/owner`
- обновляет состояние задач: назначение исполнителя, снятие, завершение

## bot_test.go

- задаёт dummy Telegram-сервер, который принимает запросы от бота и сохраняет отправленные сообщения
- переопределяет `WebhookURL` и `BotToken`, чтобы направить бота на локальный сервер
- поднимает `startTaskBot` в отдельной горутине и последовательно прогоняет набор сценариев 
- каждый сценарий имитирует отправку команды от пользователя и проверяет, что бот ответил ожидаемыми сообщениями.
